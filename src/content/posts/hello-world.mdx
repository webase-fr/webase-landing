# La vitesse n'est plus une option

Dans un monde où l'attention moyenne est inférieure à 8 secondes, la vitesse de chargement de votre site web est devenue **le critère numéro 1** de réussite.

## L'impact sur la conversion

Amazon a calculé qu'un ralentissement de **100ms** leur coûtait **1% de ventes**. Pour une PME, l'impact est proportionnellement encore plus grand. Un utilisateur qui attend plus de 3 secondes quitte le site avant même d'avoir vu votre offre.

> "La performance web est l'expérience utilisateur."

## Le SEO et Core Web Vitals

Google ne s'en cache pas : la vitesse est un facteur de classement majeur. Avec les **Core Web Vitals**, Google mesure l'expérience réelle des utilisateurs.

Si votre site est lent :
1. Vous êtes moins bien référencé.
2. Vos publicités coûtent plus cher (Quality Score).
3. Vos visiteurs partent.
